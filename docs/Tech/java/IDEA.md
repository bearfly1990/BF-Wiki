# IDEA

```
1、alt+ctrl+shift+s打开项目结构
2、点击Modules，删除已有module
3、点击加号—>import modules ---->选择项目根目录 —>选中import module from external model，选中maven（我使用的是maven）—>next,finish---->最后将module名称修改成你需要的名称
```
